---
title: "Dame un segundo"
date: 2017-10-05
---

Dame un segundo
para leerte 38 versos.
Dame un segundo
para volver a mirarte a los ojos antes de que duermas.
Dame un segundo
para volver a acariciar tus alas antes de que eches a volar.
Dame un segundo
para escapar de este mundo gris antes de que nos robe los colores.
Dame un segundo
para que miremos a la luna antes de que esta marche.
Dame un segundo
para abrazarte antes de la batalla.
Dame un segundo
para besarte antes de que gritemos "revolución".
Dame un segundo
para hacerte cosquillas antes de que estallen las trincheras.
Dame un segundo
para enredarme en tus sábanas antes de que estas se vuelvan frías.
Dame un segundo
para cantarte antes de quedarme sin voz.
Dame un segundo
para quererte antes de que mi corazón se desgaste.
Dame un segundo
para leerte
para mirarte
para acariciarte
para escaparnos
para mirar la Luna
para abrazarte
para besarte
para hacerte cosquillas
para enredarme
para cantarte
para quererte.
Dame un segundo
para engañar al tiempo
y hacerle creer que un segundo
dura tanto como 38 versos.
O quizá cuarenta si además del segundo
me das también un beso.
